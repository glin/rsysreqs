# rsysreqs

[![Build Status](https://travis-ci.org/glin/rsysreqs.svg?branch=master)](https://travis-ci.org/glin/rsysreqs)

## Quick Start

```sh
# Run tests and build binaries
make

# Run tests
make test
```

### Usage

```sh
$ rsysreqs
Usage of rsysreqs:
  -arch string
        architecture
  -d string
        use rules from this directory
  -dist string
        distribution
  -os string
        operating system
  -release string
        release
  -s string
        system requirements

$ rsysreqs-server
Usage of rsysreqs-server:
  -d string
        use rules from this directory
```
